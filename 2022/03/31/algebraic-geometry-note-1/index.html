Important questions

The textbook raised some important questions:

  • (Existence) Can we determine if V(f1,…,fs)≠∅V(f_1,\dots,f_s)\not=\emptysetV(f1​,…,fs​)=∅, i.e. do the equations f1=⋯=fs=0f_1=\cdots=f_s=0f1​=⋯=fs​=0 have common solutions?

  • (Variety Description) Can we determine if V(f1,…,fs)V(f_1,\dots,f_s)V(f1​,…,fs​) is finite, and if so, can we find all of the solutions explicitly?

  • (Variety Equivalence) Can we determine if two varieties are the same?

  • (Parametrization) Does every affine variety have a rational parametric representation? The answer is no. For example x=1−t21+t2,y=2t1+t2x=\frac{1-t^2}{1+t^2},y=\frac{2t}{1+t^2}x=1+t21−t2​,y=1+t22t​ only parametrize part of a circle.

  • (Implicitization) Given a parametric representation of an affine variety, can we find the defining equations (i.e., can we find an implicit representation)? The answer is yes. Given a parametric representation, we can always find the defining equations. Check the 3.3 in the textbook.

  • (Dimension) Can we define and determine the “dimension” of V(f1,…,fs)V(f_1,\dots,f_s)V(f1​,…,fs​)?

    • (Basis) To define “dimension”, we need to define basis. How can we define “basis”? The answer is “Gröbner basis”.

    • (Structure) The algebraic variety V(f1,…,fs)V(f_1,\dots,f_s)V(f1​,…,fs​) is just a set of points. Is there an algebraic structure behind this point set? The answer is ideal. Hilbert’s Nullstellensatz will allow us to construct a “dictionary” between geometry and algebra, whereby any statement about varieties can be translated into a statement about ideals (and conversely).

      • (Ideal Description) Can every ideal I◃k[x1,…,xn]I\triangleleft k[x_1,\dots,x_n]I◃k[x1​,…,xn​] be written as ⟨f1,…,fs⟩\langle f_1,\dots,f_s\rangle⟨f1​,…,fs​⟩ for some f1,…,fs∈k[x1,…,xn]f_1,\dots,f_s\in k[x_1,\dots,x_n]f1​,…,fs​∈k[x1​,…,xn​].

      • (Ideal Membership) If I◃k[x1,…,xn]I\triangleleft k[x_1,\dots,x_n]I◃k[x1​,…,xn​], is there an algorithm to decide whether a given polynomial lies in III?

      • (Nullstellensatz) Given I◃k[x1,…,xn]I\triangleleft k[x_1,\dots,x_n]I◃k[x1​,…,xn​], what is the exact relation between III and I(V(I))I(V(I))I(V(I))?

The basis for an ideal is not unique. We need bases with abundant properties.

Def 10[Gröbner basis]: Fix a monomial ordering. G={g1,…,gt}⊂IG=\{g_1,\dots,g_t\}\subset IG={g1​,…,gt​}⊂I is a “Gröbner basis”/“standard basis” if ⟨LT(g1),…,LT(gt)⟩=⟨LT(I)⟩\langle LT(g_1),\dots,LT(g_t)\rangle=\langle LT(I)\rangle⟨LT(g1​),…,LT(gt​)⟩=⟨LT(I)⟩.

Fix a monomial ordering. Then every ideal III has a Gröbner basis and the Gröbner basis is a basis of III.

It turns out Gröbner basis also addresses the ideal membership problem.
