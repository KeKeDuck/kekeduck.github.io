To better understand the optimization theory, let’s visualize definitions, lemma, and theorems in mind.

Def[Convex set]: A set XXX is convex if ∀x,y∈X,∀λ∈(0,1),λx+(1−λ)y∈X\forall x,y\in X,\forall\lambda\in(0,1),\lambda x+(1-\lambda)y\in X∀x,y∈X,∀λ∈(0,1),λx+(1−λ)y∈X.

Example: Any open ball in Rn\mathbb{R}^nRn is convex.

Consider a closed convex set V⊂RnV\subset\mathbb{R}^nV⊂Rn and a point x∈Rnx\in\mathbb{R}^nx∈Rn. We denote the projection of x on V, the closest v in V to x, by ΠV(x)\Pi_V(x)ΠV​(x).

Lemma 1: Assume V⊂RnV\subset\mathbb{R}^nV⊂Rn is a closed convex set and let x∈Rnx\in\mathbb{R}^nx∈Rn. Then z=ΠV(x)z=\Pi_V(x)z=ΠV​(x) iff. z∈Vz\in Vz∈V and ⟨v−z,x−z⟩≤0\langle v-z,x-z\rangle\leq 0⟨v−z,x−z⟩≤0 forall v∈Vv\in Vv∈V.

Def[Epigraph ]: An epigraph of a function f:Rn→Rf:\mathbb{R}^n\rightarrow\mathbb{R}f:Rn→R is epi f:={(x,y)∈Rn×R:y≥f(x)}\text{epi }f:=\{(x,y)\in\mathbb{R}^n\times\mathbb{R}:y\geq f(x)\}epi f:={(x,y)∈Rn×R:y≥f(x)}.

Def[Convex function (R\mathbb{R}R-valued)]: A function f:Rn→Rf:\mathbb{R}^n\rightarrow\mathbb{R}f:Rn→R is convex if epi f\text{epi }fepi f is convex.

Example: f(x)=x2f(x)=x^2f(x)=x2 is a convex function; f(x)=log⁡(x)f(x)=\log(x)f(x)=log(x) is a concave function.

Theorem 2: Assume f:Rn→Rf:\mathbb{R}^n\rightarrow\mathbb{R}f:Rn→R is continuously differentiable. Then
(i) f is convex iff. forall x,y, f(y)≥f(x)+⟨∇f(x),y−x⟩f(y)\geq f(x)+\langle\nabla f(x),y-x\ranglef(y)≥f(x)+⟨∇f(x),y−x⟩;
(ii) f is strictly convex iff. forall x≠yx\not=yx=y, f(y)>f(x)+⟨∇f(x),y−x⟩f(y)>f(x)+\langle\nabla f(x),y-x\ranglef(y)>f(x)+⟨∇f(x),y−x⟩.

Theorem 3: Assume f:Rn→Rf:\mathbb{R}^n\rightarrow\mathbb{R}f:Rn→R is twice continuously differentiable. Then
(i) f is convex if and only if its Hessian ∇2f\nabla^2 f∇2f is positive semidefinite forall x;
(ii) if forall x the Hessian is positive definite then f is strictly convex.

Lemma 4: Assume f:Rn→Rf:\mathbb{R}^n\rightarrow\mathbb{R}f:Rn→R is convex. Forall x in the interior of f domain, there exists δ\deltaδ and L such that ∣f(y)−f(x)∣≤L∣∣y−x∣∣|f(y)-f(x)|\leq L||y-x||∣f(y)−f(x)∣≤L∣∣y−x∣∣ whenever ∣∣y−x∣∣<δ||y-x||<\delta∣∣y−x∣∣<δ.

This lemma implies: we can find a cone in shape “∨\vee∨“ inside a convex set.

Def[Proper function (R\mathbb{R}R-valued)]: A function f is called proper if ∀x,f(x)>−∞\forall x,f(x)>-\infty∀x,f(x)>−∞ and ∃x,f(x)<+∞\exists x,f(x)<+\infty∃x,f(x)<+∞.

Proper convex functions (R\mathbb{R}R-valued) are important subjects in nonlinear optimization.

Def[Gradient ]: The gradient of a R\mathbb{R}R-valued function f, denoted as ∇f\nabla f∇f, is the transpose of derivative DfDfDf.

∇f:=[∂f∂x1∂f∂x2⋮∂f∂xn]\nabla f:=\begin{bmatrix}\frac{\partial f}{\partial x_1}\\\frac{\partial f}{\partial x_2}\\\vdots\\\frac{\partial f}{\partial x_n}\end{bmatrix}∇f:=⎣⎢⎢⎢⎢⎢⎡​∂x1​∂f​∂x2​∂f​⋮∂xn​∂f​​⎦⎥⎥⎥⎥⎥⎤​

To study proper convex functions we generalize “gradient” to “subgradient”.

Def[Subgradient ]: For f:Rn→R‾f:\mathbb{R}^n\rightarrow\overline{\mathbb{R}}f:Rn→R, fix a point x∈dom fx\in\text{dom }fx∈dom f, a vector ggg is a subgradient if ∀y∈Rn\forall y\in\mathbb{R}^n∀y∈Rn, we have f(y)≥f(x)+⟨g,y−x⟩f(y)\geq f(x)+\langle g,y-x\ranglef(y)≥f(x)+⟨g,y−x⟩.

Lemma 5: ∂f(x)\partial f(x)∂f(x) is a closed and convex set.

Subgradient is useful. It provides lower bound for directional derivatives.

Lemma 6: Assume that f is proper convex. ggg is a subgradient of f at x iff. ∀d∈Rn,Ddf(x)≥⟨g,d⟩\forall d\in\mathbb{R}^n,D_df(x)\geq\langle g,d\rangle∀d∈Rn,Dd​f(x)≥⟨g,d⟩.

Denote the set of all subgradients of f at x as ∂f(x)\partial f(x)∂f(x), and is called the subdifferential of f at x.

Lemma 7: Assume that f is proper convex. Then Ddf(x)⋅d∣∣d∣∣2∈∂f(x)D_df(x)\cdot\frac{d}{||d||^2}\in\partial f(x)Dd​f(x)⋅∣∣d∣∣2d​∈∂f(x).

Theorem 7: Assume that f:Rn→R‾f:\mathbb{R}^n\rightarrow\overline{\mathbb{R}}f:Rn→R is convex and x in interior of f domain. Then ∂f(x)\partial f(x)∂f(x) is a nonempty, convex, closed and bounded set. Furthermore, for every direction d∈Rnd\in\mathbb{R}^nd∈Rn one has Ddf(x)=max⁡g∈∂f(x)⟨g,d⟩D_df(x)=\max_{g\in\partial f(x)}\langle g,d\rangleDd​f(x)=maxg∈∂f(x)​⟨g,d⟩.

We say that f is subdifferentiable if forall x it has at least one subgradient.

“Subdifferentiable” is important because you don’t always expect f to be differentiable.

What happens if we drop the “proper” condition in the above theorem?

Lemma 8: If a convex function f:Rn→R‾f:\mathbb{R}^n\rightarrow\overline{\mathbb{R}}f:Rn→R is subdifferentiable at a point x, then for every d, Ddf(x)=sup⁡g∈∂f(x)⟨g,d⟩D_df(x)=\sup_{g\in\partial f(x)}\langle g,d\rangleDd​f(x)=supg∈∂f(x)​⟨g,d⟩.
Moreover, if Ddf(x)<∞D_df(x)<\inftyDd​f(x)<∞ then we can replace “sup” with “max”, i.e. the supremum is attained.

What happens if f is differentiable?
